# Example Projects

This directory contains sample project files compatible with Crom-FrameFlow.

## Contents

- **[basic-demo.json](./basic-demo.json)**: A simple project with one video track and a title overlay. Uses a public sample video URL.

## How to use

1. Open FrameFlow.
2. Click **Import Project** (if available) or drag and drop the JSON file into the editor.
   _(Note: JSON import feature must be enabled in the app)_

## Structure

Project files use a JSON structure defining:

- `timeline`: Tracks and Clips.
- `resources`: Assets mapping (optional).
- `metadata`: Author/Date info.
